Virtual Extension Number

This is a 3 or 4 digit number that is used in the PBX to route calls to your virtual extension DID.

Forward Number

This is the number that will be called when someone enters the extension number.

Timeout (seconds)

How many seconds to ring before the call fails and continues to the next route in the system.

Timeout Destination

The destination in the PBX that will handle the call after the given timeout period has expired.
